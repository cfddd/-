## 功能概览
Sequence 用于追踪 RingBuffer 中事件的位置信息：
- 生产者序列：当前已发布到哪个位置
- 消费者序列：当前已处理到哪个位置
- 屏障序列：依赖关系中的同步点
## Value
```java
class LhsPadding
{
    protected long p1, p2, p3, p4, p5, p6, p7;
}

class Value extends LhsPadding
{
    protected volatile long value;
}

class RhsPadding extends Value
{
    protected long p9, p10, p11, p12, p13, p14, p15;
}
```
设计目的：
- 缓存行隔离：确保 value 字段独占 64 字节缓存行
- 避免伪共享：防止多个 Sequence 对象的 value 字段在同一缓存行
- 性能优化：减少 CPU 缓存失效，提升并发性能
